# .po-File-reader-from-scratch
## 从零开始编写一个 .po 文件解析器

#### An tutorial about building an .po file reader for machine-translate backend.



这是一套关于 **Python 自然语言处理** 的实验教程。

实验项目是实现 .po 文件的翻译自动化。项目分为 **文件解析** 和 **机器翻译** 两个部分，此教程主要讲解了 **.po 文件解析器的实现**。
